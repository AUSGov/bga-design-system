Anatomy

Light pathway elements

Title

The title of the linked page.

Default - Heading 3 (desktop)

    font-family: OpenSans-Semibold;
    font-size: 21px;
    color: #333333;
    letter-spacing: 0;
    line-height: 28px;
}
h3:hover {
    color: #254F90;
    text-decoration: underline;
}
Default - Heading 3 (mobile)
h3 {    
    font-family: OpenSans-Semibold;
    font-size: 21px;
    color: #333333;
    letter-spacing: 0;
    line-height: 30px;
}

Layouts

Page width

Light pathway components should only be used in full width pages.

Equal width layout

Desktop: Three equal width components in a row.

Tablet: Two equal width components in a row.

Mobile: One equal component in a row.
