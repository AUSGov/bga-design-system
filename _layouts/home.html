Introducing the design system

How to use the design system

The design system provides a framework for selection of page patterns and components to deliver an intended user experience. However, is not a defined set of fixed templates. It relies on design decisions being made in the context of individual pages content and the desired user experience.

Foundations

Global rules and elements which govern the appearance, structure and behaviour of components.

Components

Components are the building blocks of the site. They are a modular system of parts which can be added to page patterns.
