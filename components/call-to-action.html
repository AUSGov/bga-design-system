Guidelines for use

Purpose

Used as a single main action on a page. Typically used on transaction pages and includes a summary of the next steps and an optional list of prerequisite actions.

Layouts

Page width

Call to action components can be used in full width or two-third width pages.

Note: In full width layouts this component is centered on the page with a maximum width of 620px.
