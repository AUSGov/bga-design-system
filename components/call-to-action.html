Guidelines for use

Purpose

Standard call to action

Used as a single main action on a page. Typically used on internal program pages and transaction pages and includes a summary of the next steps and an optional list of prerequisite actions.

When to use these components

Standard call to action

The Standard CTA should be used when the user must complete an important task (level 1 priority task).

  • only one per page
  • description text must be included so the user knows what they need to do, can be bullet points, ticks or a sentence
