Layouts

Page width

Image pathway list components can be used in full width and 2/3rd width pages.

Equal width layout - full width pages

Desktop: Three equal width components in a row.

Tablet: Two equal width components in a row.

Mobile: One equal component in a row.

Equal width layout - 2/3rd width pages

Desktop & tablet: Three equal width components in a row.

Mobile: One equal component in a row.
