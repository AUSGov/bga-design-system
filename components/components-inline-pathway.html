Overview

Purpose

The inline pathway provides links related to specific page content.

It can be used singularly within the context of written content if it is a high value step in a process. It can also be used at the end of the page to provide links to explore the subject deeper.

Appearance

The inline pathway can link to either internal site pages.

The inline pathway can also link to external sites.

Layouts

Page width

Inline pathway components can be used in full width or two-third width pages.

Note: In full width layouts this component is centered on the page with a maximum width of 620px.
