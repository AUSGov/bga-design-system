Approved icons

Icon colour MUST be matched with the notification box colour:

  • The blue information icon pairs with the blue notification box
  • The green confirmation icon pairs with the green notification box
  • The red warning icon pairs with the red notification box

Guidelines for use

Purpose:

Notifications draw the users attention to specific information on a page or during a process.

When to use this component:

  • Often used in tools or forms to provide information or feedback in response to a user action.
