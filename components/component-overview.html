Component types

Components are broken into three main categories:

  • Pathway components
  • Content components
  • Menu components

Pathway components

Pathways are the primary method of in-page navigation, they promote a link to another page or website as a tile within the context of the page. They include include a summary of content and purpose.

Displaying multiple pathways as a list

Components that include list in the name display a grid of multiple pathways, each tile is added as an item in the one component. The display of every component is treated the same across the entire list, eg if you choose to hide the description it will be hidden across every component.

Content components

Content components format in-page copy and imagery to present the content as clearly as possible for the user.

Menu components

Menu components enable the user to easily navigate through business.gov.au site or tools.
