Overview

Purpose

Generally used as a lightweight option within index pages alongside hero, standard or featured pathways to build page hierarchy.

Appearance

Light pathway

The light pathway is an in-page link to another page on the site.

Layouts

Page width

Light pathway components should only be used in full width pages.

Equal width layout

Note: Components have a 32px horizontal gutter between them but zero padding between rows.

Desktop: Three equal width components in a row.

Tablet: Two equal width components in a row.

Mobile: One equal component in a row.
