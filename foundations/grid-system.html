Vertical grid

The business.gov.au website is based on a 12 column grid with five major breakpoints.

Desktop HD

Minimum screen width: 1600px;
Body width: 1208px
Column width: 64px;
Gutter width: 40px;

Desktop

Body width: 976px
Column width: 52px;
Gutter width: 32px;

Laptop

Minimum screen width: 900px;

Column width: 40px;
Gutter width: 24px;

Tablet

Minimum screen width: 480px;
Body width: 416px
Column width: 20px;
Gutter width: 16px;

Mobile

Minimum screen width: 320px;
Body width: 272px
Column width: 8px;
Gutter width: 16px;
