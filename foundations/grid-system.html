Vertical grid

The business.gov.au website is based on a 12 column grid with five major breakpoints.

Desktop HD

Minimum screen width: 1600px;
Body width: 1208px
Column width: 64px;
Gutter width: 40px;

Desktop

Minimum screen width: 1200px;
Body width: 976px
Column width: 52px;
Gutter width: 32px;

Laptop

Minimum screen width: 900px;
Body width: 744px
Column width: 40px;
Gutter width: 24px;

Tablet

Minimum screen width: 480px;
Body width: 416px
Column width: 20px;
Gutter width: 16px;

Mobile

Minimum screen width: 320px;
Body width: 272px
Column width: 8px;
Gutter width: 16px;

Horizontal spacing

Horizontal spacing is based on 8px increments. Unless otherwise stated the standard horizontal space between two elements is 32px.

Heading 1

0px above, 32px below

Heading 2

24px above, 32px below

Heading 2 is often used with a short yellow horizontal line underneath which has the following padding:
0px above, 32px below

Heading 3

16px above, 32px below

Heading 4

16px above, 16px below

Heading 5

0px above, 16px below

Paragraph

0px above, 32px below

List

0px above, 16px below (except for the last list item which has 32px below)

All block elements

0px above, 32px below

Includes all block design elements such as:

  • pathway tiles
  • tables
  • callout boxes
  • calls to actions boxes
  • images
  • accordions
  • media player
