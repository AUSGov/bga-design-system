Vertical grid

The business.gov.au website uses the bootstrap 12 column grid .

Grid breakpoints

Extra small Small Medium Large Extra large
Minimum screen width 320px 576px 768px 992px 1200px
Maximum container width none 540px 720px 960px 1140px
Gutter - 30px 30px 30px 30px

Horizontal spacing

Horizontal spacing is based on 8px increments. Unless otherwise stated the standard horizontal space between two elements is 32px.

Heading 1

0px above, 32px below

Heading 2

24px above, 32px below

Heading 2 is often used with a short yellow horizontal line underneath which has the following padding:
0px above, 32px below

Heading 3

16px above, 32px below

Heading 4

16px above, 16px below

Heading 5

0px above, 16px below

Paragraph

0px above, 32px below

List

0px above, 16px below (except for the last list item which has 32px below)

All block elements

0px above, 32px below

Includes all block design elements such as:

  • pathway tiles
  • tables
  • callout boxes
  • calls to actions boxes
  • images
  • accordions
  • media player
