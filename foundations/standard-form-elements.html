Warning notification

Design view:

Mobile design

Desktop designs can be viewed on screens wider than 650px.

Warning alerts contain information on errors that the user needs to know.

  • Critical error that needs to be fixed
  • Another thing that needs to be fixed before you can continue

Note: An error summary can be used when a form that contains errors is submitted. The error summary should never replace error states and messaging on each form field.
