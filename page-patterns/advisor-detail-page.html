1Heading (page title)

A direct, short title that tells the user who the adviser is.

  • Maximum character limit of 50.
2Body text

A short paragraph that details what the adviser offers (e.g. support, services). Specific services are to be provided in a list.

Guidelines for use

Purpose

An adviser detail page is a single page detailing one adviser per page.

It should always include:

  • Any relevant contact information
  • Brief description about the skills and background of the adviser (try to limit to paragraphs to keep it consistent with advisers across all areas)
  • A maximum 90 character metadata description for search (this does not appear on the page)
