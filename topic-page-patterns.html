Navigation

Navigation page pattern description

Category page

A page pattern for navigation to business topic pages.

Topic page

A page pattern for navigation to pages under a business topic.

Business topics

Business topics page pattern description

Resource article page

A short, sharp page styled like an article that helps a user with a quick, simple topic or task.

101 page

A single page that provides simple facts to help a user understand the basics of a business topic.

Transaction action page

A page that provides the user with enough information to complete a single transaction-based task confidently and quickly.

Tutorial page

Provides the user with steps and actionable guidance to help them complete a more complicated task or understand a more complicated topic.

Guides

Guides page pattern description

Guides page

A navigation pattern that pulls together information from different areas of the site.

Checklist page

A checklist pattern for presenting an end to end journey in logical steps.

Grants and programs

Grants and programs page pattern description

Internal grant landing page

A single page providing information about grant amounts, eligibility, the application process and key documents.

External grant landing page

A single page detailing info about external grants and external contact information.

Grant recipients page

A single page listing successful grant recipients, usually in table form.

Customer stories page

Is a home page for all program customer stories.

Customer story (single) page

Is a single page that provides an article about how successful grant recipients have used their grant to improve their business.

Advisors

Advisors page pattern description

Advisor detail page

An adviser detail page is a single page detailing one adviser per page.

Events

Events page pattern description

Events page

A single page listing the details about an event.

News

News page pattern description

News page

A short, sharp page that is article styled that provides timely information or action.
